Execption: it is an abnormal condition which terminates the program unexpecdely/ the flow of execution.

to handle such types of situation we use exception handling

after exception handling flow of programm will be normal


try, except, finally, else, raise - these keywords are use in exception handling


try - where the devloper feels there can be an exception, that block of code will put into try.

except - when the exception occured in try block then except block will be execute.
except is use to diaplay the message related to exception.
